# Link2Qr Crafter

Create QR codes instantly from links with a fast, modern, and user-friendly desktop experience.

Link2Qr Crafter is built for creators, developers, students, businesses, and everyday users who want a simple way to turn links into QR codes.

---

## ✨ Features

- ⚡ Instant QR code generation from links
- 🔗 Convert website URLs into QR codes
- 🎨 Clean and modern interface
- 💾 High-quality QR image export
- 📱 Perfect for websites, social media, business cards, and sharing
- 🌙 Lightweight and smooth performance
- 🖥️ Desktop-focused experience
- 🔒 Privacy focused — your data stays private and is never saved anywhere

# 🛠️ Installation & Setup (Linux AppImage)

Your download comes as a ZIP file containing the AppImage.

## Step 1 — Extract ZIP File

Right click the ZIP file and select:

```bash
Extract Here
```

OR use terminal:

```bash
unzip Link2QrCrafter.zip
```

---

## Step 2 — Give Execute Permission

Open terminal inside extracted folder and run:

```bash
chmod +x Link2Qr_Crafter-x86_64.AppImage
```

---

## Step 3 — Launch Application

Double click the AppImage

OR run:

```bash
./Link2Qr_Crafter-x86_64.AppImage
```

---

## ⚠️ If AppImage Does Not Open

Install FUSE support:

### Ubuntu / Linux Mint

```bash
sudo apt install libfuse2
```

Then run the AppImage again.
